## 🛠 Features

- Private Storage: Information stays on your machine.
- Text Extraction: The tool reads text from images automatically.
- Flexible Search: You find files even if you make typos.
- Automatic Updates: New screenshots appear in your search results without manual work.
- Duplicate Removal: The software ignores exact copies.
- Modern Look: The interface uses a clean, dark design.

## 📂 Setting Up Folders

When you open the application for the first time, you must tell it where your screenshots live. Click the "Settings" button in the corner of the window. Select the button labeled "Add Folder" and choose the folder where your computer saves screenshots. The software creates an index of your files. This takes a few minutes if you have many images.

## 🔍 Using the Search Bar

Type words into the main box at the top of the window. The engine looks through the text found inside your images. If you search for "receipt," the application displays every screenshot that contains that word. You do not need to press enter, as the list updates while you type.

## 🧹 Managing Your Library

The software checks for new images every time you launch it. If you move a screenshot to a new folder, the search engine removes the old record. You can clear the search history by clicking the "Reset" button in the menu. This action removes the internal index but keeps your original image files safe on your disk.

## 💡 Frequently Asked Questions

Does the app send data to the internet?
No. The application performs all tasks on your local computer. It does not use external servers.

Why does the search take time?
The application reads the text inside images the first time it sees them. Future searches run instantly once the initial scanning finishes.

How do I remove the app?
Open your Windows Settings, go to Apps, find the name in the list, and select Uninstall. This removes the program from your system.

What happens if I have bad lighting in a screenshot?
The software uses high-quality text extraction tools. It reads clear text easily. It may struggle with blurry images or handwriting.

Can I watch multiple folders?
Yes. Open the settings menu and add as many folders as you want. The application monitors all selected locations for updates.

Does it support other languages?
The current version focuses on English text. Future updates will include support for other languages.

Is the software safe?
The code is open for review on GitHub. It performs no actions outside your assigned folders.

What is the database file?
The program creates a small file to store references to your images. Do not move or delete this file while the application runs.

Do I need to keep the app open to index files?
Yes. The file watcher starts when you open the application. It stops if you close the window.

Why is my file not appearing?
Ensure the image is in a watched folder. Wait a few moments for the background scanner to process new content.

How do I report an issue?
Navigate to the repository page and click the Issues tab to share feedback or report errors.
